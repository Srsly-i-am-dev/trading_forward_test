Build Multi-Indicator Forward Testing System (TradingView → cTrader)
Objective

Build a local system that:

Receives TradingView webhook alerts

Executes trades on cTrader (IC Markets) (placeholder initially)

Logs all signals and trades

Tracks indicator performance per pair

Displays results in a dashboard

System must support:

10–15 indicators

8–10 forex pairs

~150 strategy streams

No VPS. Runs locally.

System Architecture

TradingView Indicators
→ TradingView Alerts
→ ngrok public webhook
→ Local Python Webhook Server
→ Trade Executor
→ Trade Logger
→ SQLite Database
→ Analytics Dashboard

Database Module

Create:

database/db.py

Code:

import sqlite3

DB_PATH = "logs/trades.db"


def init_db():
    conn = sqlite3.connect(DB_PATH)
    cursor = conn.cursor()

    cursor.execute("""
    CREATE TABLE IF NOT EXISTS trades(
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        indicator TEXT,
        symbol TEXT,
        action TEXT,
        price REAL,
        timestamp DATETIME DEFAULT CURRENT_TIMESTAMP
    )
    """)

    conn.commit()
    conn.close()


def log_trade(indicator, symbol, action, price):

    conn = sqlite3.connect(DB_PATH)
    cursor = conn.cursor()

    cursor.execute("""
    INSERT INTO trades(indicator, symbol, action, price)
    VALUES (?, ?, ?, ?)
    """, (indicator, symbol, action, price))

    conn.commit()
    conn.close()

Webhook Server

Create:

server/webhook_server.py

Code:

from flask import Flask, request, jsonify

from executor.ctrader_executor import execute_trade
from database.db import log_trade

app = Flask(__name__)


@app.route("/webhook", methods=["POST"])
def webhook():

    data = request.json

    indicator = data.get("indicator")
    symbol = data.get("symbol")
    action = data.get("action")

    print("Received signal:", data)

    price = execute_trade(data)

    log_trade(indicator, symbol, action, price)

    return jsonify({"status": "ok"})


if __name__ == "__main__":
    app.run(host="0.0.0.0", port=5000)
Dashboard

Create:

dashboard/dashboard.py

Code:

import sqlite3
import pandas as pd
import streamlit as st

conn = sqlite3.connect("logs/trades.db")

df = pd.read_sql("SELECT * FROM trades", conn)

st.title("Forward Testing Dashboard")

st.subheader("All Signals")

st.dataframe(df)

st.subheader("Signals per Indicator")

indicator_counts = df["indicator"].value_counts()

st.bar_chart(indicator_counts)

st.subheader("Signals per Symbol")

symbol_counts = df["symbol"].value_counts()

st.bar_chart(symbol_counts)
